Introdução

Essas são as três principais linguagens utilizadas no front-end da nossa aplicação, ou seja, são utilizadas do lado do cliente no próprio navegador.

História

Na década de 1990, o compartilhamento de documentos entre empresas era complexo e demorado. Em 1991, Tim Berners-Lee concebeu o HTML para viabilizar essa troca de forma mais eficaz. A World Wide Web (WWW) surgiu logo após, globalizando o uso do HTML. O CSS foi criado em 1995 para separar a parte estética do HTML, que estava crescendo rapidamente. Quanto ao JavaScript, sua origem envolveu uma competição entre a Microsoft e a Netscape. A Netscape criou o JavaScript para ser executado nos navegadores, enquanto a Microsoft desenvolveu o JScript. A padronização levou à criação do ECMAScript, antigo nome do JavaScript, que se tornou uma linguagem poderosa para interatividade web.

HTML

O HTML, abreviação de Hyper Text Markup Language, é uma linguagem de marcação de hipertexto. Diferentemente de uma linguagem de programação, o HTML organiza os elementos em páginas web usando tags (etiquetas) para definir estruturas como cabeçalhos, parágrafos e imagens. Essa linguagem cria a base da página, permitindo a marcação de conteúdo semelhante a um esqueleto.

CSS

O Cascading Style Sheets (CSS) é uma linguagem que separa a aparência da estrutura em páginas da web. Isso é feito usando seletores, propriedades e valores para estilizar elementos HTML. O termo "folha de estilo em cascata" refere-se à hierarquia de estilos que podem ser aplicados em camadas. O CSS pode ser inserido no HTML usando <style>, como atributos ou em arquivos separados, que podem ser vinculados usando <link>. O uso de vários arquivos CSS é possível, mas deve ser feito com organização para evitar confusão.

JavaScript

JavaScript é a linguagem de programação da tríade do desenvolvimento front-end, junto com HTML e CSS. Ele adiciona interatividade às páginas da web, permite atualizações dinâmicas e processamento de dados. Pode ser usado tanto no lado do cliente quanto do servidor. Com JavaScript, é possível criar elementos animados, caixas de alerta e funcionalidades dinâmicas em sites. Pode ser inserido no HTML usando tags <script> ou em arquivos separados, e até mesmo usado com tecnologias do lado do servidor como o Node.js.

Papel na aplicação

→HTML: linguagem de marcação utilizada para estruturar os elementos da página, como parágrafos, links, títulos, tabelas, imagens e até vídeos.

→CSS: linguagem de estilos utilizada para definir cores, fontes, tamanhos, posicionamento e qualquer outro valor estético para os elementos da página.

→Javascript: linguagem de programação utilizada para deixar a página com mais movimento, podendo atualizar elementos dinamicamente e lidar melhor com dados enviados e recebidos na página.
